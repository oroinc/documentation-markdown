# Create a Tax Rate

<!-- begin -->

To create a new tax rate:

1. Navigate to **Taxes > Taxes** in the main menu.
2. Click **Create Tax**.
3. Fill in **Code**, **Description**, and **Rate (%)**.
4. Click **Save and Close**.

<!-- stop -->
