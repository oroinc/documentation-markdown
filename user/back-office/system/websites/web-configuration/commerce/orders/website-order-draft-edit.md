# Configure Order Draft Edit Mode Settings per Website

#### NOTE
The Order Draft Edit Mode feature is available as of OroCommerce version 6.1.9 and can be enabled [globally](../../../../configuration/commerce/orders/global-order-draft-edit.md#system-configuration-orders-draft-edit-mode), [per organization](../../../../user-management/organizations/org-configuration/commerce/orders/organization-order-draft-edit.md#configuration-commerce-orders-draft-edit-mode-org), and per website.

To enable this feature per website:

1. Navigate to **System > Websites** in the main menu.
2. For the necessary website, hover over the <i class="fa fa-ellipsis-h fa-lg" aria-hidden="true"></i> **More Options** menu to the right of the necessary website and click <i class="fas fa-cog" aria-hidden="true"></i> to start editing the configuration.
3. Select **Commerce > Orders > Order Draft Edit Mode** in the menu to the left.

#### NOTE
For faster navigation between the configuration menu sections, use [Quick Search](../../../../configuration/quick-search.md#user-guide-system-configuration-quick-search).

![Order Draft Edit Mode configuration settings per website](user/img/system/config_commerce/order/order-draft-edit-website.png)
1. In the **Order Draft Edit Mode** section:
   1. Clear the **Use Organization** checkbox to adjust the settings per specific website.
   2. Select the checkbox next to **Enable Order Draft Edit Mode** to enable the feature.
   3. Once enabled, the order edit page uses draft-based editing behavior for all order updates. For more details on the order draft editing behavior, please refer to the [related article](../../../../configuration/commerce/orders/global-order-draft-edit.md#system-configuration-orders-draft-edit-mode)
2. Click **Save Settings**.
