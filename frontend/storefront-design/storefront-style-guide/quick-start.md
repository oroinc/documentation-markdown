## Prerequisites

To start using the OroCommerce Storefront Style Guide, you need to download and install:

* <a href="https://www.sketch.com/get/" target="_blank">Sketch app</a> - A graphical editor that enables you to design user interfaces and prototypes. The <a href="https://www.sketch.com/docs/" target="_blank">user guide</a> helps learn general principles of usage and quickly understand the process of creating the Storefront theme design using the Style Guide.
* <a href="https://fonts.google.com/specimen/Lato" target="_blank">Lato</a> - A font that is applied in the Storefront theme.
* <a href="https://fontawesome.com/v4.7.0/" target="_blank">FontAwesome 4.7</a> - A font for icons rendering in the Storefront theme.

#### HINT
Sketch is developed for the MAC OS users. Please read the requirements before the installation. You can also open the Storefront Style Guide with Figma or Adobe XD. For that, read <a href="https://help.figma.com/hc/en-us/articles/360040514273-Import-files-from-Sketch" target="_blank">Import files from Sketch into Figma</a> or <a href="https://helpx.adobe.com/xd/user-guide.html/xd/kb/open-sketch-files-in-xd.ug.html" target="_blank">Supported features when you open Sketch files in Adobe XD</a>.
